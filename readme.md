# HaymanBacktester

## 1. 설명
대신증권 api 기반의 백테스팅 프로그램
        원하는 종목을 넣어 자신이 원하는 기술적 분석 기반의 전략으로 백테스팅을 실시할 수 있습니다.
        단순 매수후 보유 수익 그래프와 비교하여 성과가 어느정도인지 알 수 있습니다.

## 2. 개발 환경
python 3.6
anaconda 5.2.0

## 3. branch 
- master : 안정 프로그램
- develop : 개발중인 프로그램
- feature : 개별 기능
- hotfix : 프로그램 버그잡기
